AWK Command in Linux with Examples

Last Updated: 15 Jan, 2025

The AWK command in Linux is a versatile scripting language used for text processing, data extraction, and report generation. It scans files line by line, matches patterns, and performs actions on matched lines.

Table of Contents

  • Introduction to AWK
  • Syntax and Options
  • Built-in Variables

  • Frequently Asked Questions

Introduction to AWK

AWK is named after its creators: Alfred Aho, Peter Weinberger, and Brian Kernighan. It enables programmers to write concise and effective scripts for:

  • Scanning a file line by line
  • Splitting lines into fields
  • Comparing lines or fields to patterns
  • Performing actions on matched lines

Syntax and Options

The basic syntax of the AWK command is:

awk [options] 'pattern { action }' input-file > output-file

Commonly used options include:

Option Description
-f program-file Reads the AWK program source from the specified file.
-F fs Specifies the field separator (default is whitespace).

Built-in Variables

AWK provides several built-in variables for efficient text processing:

Variable Description
NR Current record number (line number).
NF Number of fields in the current record.
FS Field separator (default is whitespace).
OFS Output field separator.
ORS Output record separator (default is newline).

Examples of AWK Commands

1. Print All Lines

awk '{print}' employee.txt

Output:

ajay manager account 45000
sunil clerk account 25000
...

2. Print Lines Matching a Pattern

awk '/manager/ {print}' employee.txt

Output:

ajay manager account 45000
varun manager sales 50000
...

3. Print Specific Fields

awk '{print $1, $4}' employee.txt

Output:

ajay 45000
sunil 25000
...

4. Display Line Numbers

awk '{print NR, $0}' employee.txt

Output:

1 ajay manager account 45000
2 sunil clerk account 25000
...

Frequently Asked Questions

What is the AWK command used for?

AWK is used for text processing, including pattern matching, data extraction, and report generation.

What does awk '{ print $1 }' do?

It prints the first field of each line from the input file.

What is the full form of AWK?

AWK is named after its developers: Aho, Weinberger, and Kernighan.
