Cut Command in Linux with Examples

Last Updated: 12 Jul, 2024

The cut command in Linux is used to extract sections from each line of a file and write the result to standard output. It can be used to cut parts of a line by byte position, character, and field.

Table of Contents

  • Syntax of cut Command
  • Options Available in cut Command
  • Practical Examples of cut Command
  • Frequently Asked Questions on cut Command in Linux

Syntax of cut Command

The basic syntax of the cut command is:

cut OPTION... [FILE]...

Where:

  • OPTION specifies the desired behavior.
  • FILE represents the input file.

Options Available in cut Command

Here is a list of the most commonly used options with the cut command:

Option Description
-b, --bytes=LIST Selects only the bytes specified in LIST (e.g., -b 1-3,7).
-c, --characters=LIST Selects only the characters specified in LIST (e.g., -c 1-3,7).
-d, --delimiter=DELIM Uses DELIM as the field delimiter character instead of the tab character.
-f, --fields=LIST Selects only the fields specified in LIST, separated by the delimiter character (default is tab).
-n Do not split multi-byte characters (no effect unless -b or -c is specified).
--complement Invert the selection of fields/characters. Print the fields/characters not selected.

Practical Examples of cut Command

Extract Specific Bytes (-b) Using cut Command

To extract specific bytes, use the -b option:

cut -b 1,2,3 state.txt

Example with a range of bytes:

cut -b 1-3,5-7 state.txt

Cut by Character (-c) Using cut Command

To cut by character, use the -c option:

cut -c 2,5,7 state.txt

Example of cutting the first seven characters:

cut -c 1-7 state.txt

Cut by Field (-f) Using cut Command

To cut by field, use the -f option. If fields are separated by a space, you can also use the -d option:

cut -d " " -f 1 state.txt

Frequently Asked Questions on cut Command in Linux

How do I use the cut command to extract specific columns from a file?

Example: To extract the first and third columns from a CSV file named data.csv:

cut -d',' -f1,3 data.csv

Can I use cut to extract a range of characters from each line?

Yes, you can. To extract characters 5 to 10 from each line of a file named text.txt:

cut -c5-10 text.txt

Conclusion

The cut command in Linux is a versatile tool for extracting specific sections from files based on byte, character, or field. By using different options like -b, -c, and -f, you can slice text and extract exactly what you need.
