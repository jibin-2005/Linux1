Sed Command

The sed (Stream Editor) command in Unix/Linux is a powerful utility used to process and manipulate text in files. It can perform operations like searching, find-and-replace, insertion, and deletion, making it an efficient tool for text management tasks.

Description:

sed allows you to manipulate text directly in files or streams without opening them in an editor. It supports regular expressions, enabling complex pattern matching and automated edits.

Syntax:

sed [OPTIONS] 'SCRIPT' [INPUTFILE...]

Commonly Used Options:

  • -i: Edit the file in place without printing to the console.
  • -n: Suppress automatic printing of lines.
  • -e: Execute multiple commands.
  • -f: Read commands from a file.
  • -r: Enable extended regular expressions.

Examples:

1. Replacing or Substituting Strings:

Replace the first occurrence of "unix" with "linux":

sed 's/unix/linux/' geekfile.txt

Output:

linux is great os. unix is opensource. unix is free os.
learn operating system.
linux linux which one you choose.
linux is easy to learn.unix is a multiuser os.Learn unix .unix is a powerful.

2. Replace All Occurrences:

Use the /g flag for global replacement:

sed 's/unix/linux/g' geekfile.txt

3. Replace on Specific Line:

Replace "unix" with "linux" only on the third line:

sed '3 s/unix/linux/' geekfile.txt

4. Deleting Lines:

  • Delete the 5th line: sed '5d' geekfile.txt
  • Delete lines from 3 to 6: sed '3,6d' geekfile.txt
  • Delete lines containing "abc": sed '/abc/d' geekfile.txt

5. Using Multiple Commands:

Combine commands with -e:

sed -e 's/unix/linux/' -e '/opensource/d' geekfile.txt

Conclusion:

The sed command is a versatile and powerful tool for text processing, enabling efficient automation of editing tasks for files and streams.
