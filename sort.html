Sort Command

The sort command is used to arrange the contents of a text file or input in a specified order, either ascending or descending, alphabetically or numerically.

Description:

The sort command is typically used in Linux and Unix-based systems to sort text files in a particular order. By default, it sorts lines in ascending order based on ASCII value. The sort command has multiple options and is highly customizable depending on the type of data being sorted (e.g., numerical, alphabetic, reverse order, etc.).

Syntax:

sort [options] [file(s)]

Usage Examples:

  • Sort Alphabetically: sort file.txt
  • Sort Numerically: sort -n file.txt
  • Sort in Reverse Order: sort -r file.txt
  • Sort Based on Specific Column: sort -k 3 employee_data.txt
  • Sort and Remove Duplicates: sort -u file.txt

Commonly Used Options:

  • -o: Specifies the output file for sorted data. sort -o output.txt input.txt
  • -n: Sorts data numerically. sort -n numbers.txt
  • -r: Sorts data in reverse order (descending). sort -r file.txt
  • -k: Sorts based on a specific column. sort -k 2 file.txt
  • -u: Removes duplicate lines. sort -u file.txt
  • -M: Sorts by month names. sort -M month_list.txt

Example - Sorting Numerical Data:

Input:
  5
  2
  8
  1

Command: sort -n

Output:
  1
  2
  5
  8

Sorting with Mixed Case:

By default, uppercase letters are sorted before lowercase letters. You can use the -f option to sort without considering case (case-insensitive sorting).

Example - Case-Insensitive Sort:

Input:
  Banana
  apple
  Cherry
  apple

Command: sort -f

Output:
  apple
  apple
  Banana
  Cherry

Sorting Based on Specific Column:

You can sort data based on specific columns using the -k option. This is useful for sorting structured data like CSV or TSV files.

Example - Sorting by Column 2:

Input (employee_data.txt):
  Name, Age, Department
  Alice, 30, HR
  Bob, 25, IT
  Charlie, 35, Marketing

Command: sort -t, -k2 employee_data.txt

Output:
  Bob, 25, IT
  Alice, 30, HR
  Charlie, 35, Marketing

Sorting Files and Directories:

You can also use the sort command to sort the list of files and directories in a specific order.

Example - Sorting Files by Name:

Command: ls | sort

Output:
  a.txt
  b.txt
  c.txt

Advanced Sorting - Multiple Keys:

Sort data based on multiple columns using the -k option. For example, sorting by both last name and first name in a list of names.

Example - Sorting by Multiple Columns:

Input (names.txt):
  Alice, Johnson
  Bob, Smith
  Charlie, Brown

Command: sort -t, -k2,2 -k1,1 names.txt

Output:
  Charlie, Brown
  Alice, Johnson
  Bob, Smith

Additional Options:

  • -b: Ignores leading spaces.
  • -f: Ignores case when sorting.
  • -t: Specifies a custom delimiter (e.g., for CSV or TSV files).
  • -c: Checks whether the input is already sorted.

Conclusion:

The sort command is a versatile and powerful tool in the Linux/Unix toolkit. It can be used to sort text files, numeric data, and even directories, making it an essential tool for anyone working with large datasets or performing data manipulation tasks.
